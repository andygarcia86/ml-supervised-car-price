# Car Price Prediction Dataset

## Source

This dataset was obtained from Kaggle:

**URL:** https://www.kaggle.com/datasets/jacksondivakarr/sample34

**Author:** Jackson Divakar

## Description

This dataset is used for training a supervised machine learning model to predict car prices based on various vehicle attributes.

## Usage

This dataset is used in the Car Price Predictor project to train and evaluate ML models for estimating vehicle prices.
